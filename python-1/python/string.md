# String

#### type

```python
>>> type("hello")
<class 'str'>

>>> type('hello')
<class 'str'>
```

#### line breaker

```bash
>>> print('''first
... second
... third''')
first
second
third
>>>
```

#### length, index, slice

```bash
>>> word='python'
>>> len(word)
6

>>> word[0]
'p'

>>> word[0:6]
'python'

>>> word[0:]
'python'

>>> word[:5]
'pytho'

>>> word[:-2]
'pyth'
```
